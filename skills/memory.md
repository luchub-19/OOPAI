**memory tool**
- công cụ memory tool này (call bằng "memory_tool") được sử dụng khi
- lưu trữ dữ liệu: nếu bạn thấy dữ liệu mà nên cần lưu trữ thì sử dụng tool này
- tìm kiếm bộ nhớ: nếu mà bạn cảm thấy sử dụn tool này đê search, hãy dùng tool này. Lưu ý nên kiểm tra nhiều keyword để tăng khả năng tìm được dữ liệu (tầm 7 keyword)vd : "{"action": "search", "data": "keyword 1, keyword 2,keyword 3, keyword 4,keyword 5, keyword 6,keyword 7"}"
